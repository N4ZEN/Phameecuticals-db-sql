## About The Project

This is a secure database for  a pharmaceutical company that specializes in viral treatments, vaccinations, and medical research. 
The database is created using Microsoft SQL Server Version: 15.0.2000.5. This database consists of various features such as q 
Database Auditing Environment, Password Policies and Authentication, Logon triggers, Encryption and Backup and 
Restore operations.
